# 部署文档目录索引

本文档用于快速了解 `部署文档` 目录中每个文件的作用。

## 文件总览

### 1. `README.md`
目录总入口，概述项目、快速开始方式和各文档用途。

### 2. `DEPLOYMENT.md`
正式部署手册，包含 Windows 本地启动、Docker 启动、环境变量说明和常见排错。

### 3. `.env.example`
环境变量示例文件，用于复制后填写真实配置。

### 4. `requirements.txt`
基础依赖清单，便于快速安装项目运行所需包。

### 5. `requirements-lock.txt`
锁定版本的依赖清单，用于稳定复现部署环境。

### 6. `notes-streaming.txt`
流式回答功能的开发记录与实现备忘。

### 7. `RAG性能优化说明.md`
RAG 检索链路性能优化总结，说明优化了什么、为什么这么做、带来了什么收益。

### 8. `模型作用说明.md`
说明项目中各模型/组件的职责，以及它们在 RAG 流程中的位置。

## 建议阅读顺序

1. `README.md`
2. `DEPLOYMENT.md`
3. `.env.example`
4. `RAG性能优化说明.md`
5. `模型作用说明.md`

如果你要部署项目，优先看 `DEPLOYMENT.md`。
如果你要理解系统设计，优先看 `RAG性能优化说明.md` 和 `模型作用说明.md`。
