### RAG 优势

1. **事实准确**：所有回答均基于招股说明书原文，附带页码引用，可追溯验证
2. **无幻觉**：10 道题零编造，严格基于检索内容回答

4. **可追溯**：每条回答标注来源页码，便于人工复核

### 纯 LLM 的问题

1. **严重幻觉**：10 道题中 8 道编造了具体数据或名称（如编造注册资本 1.03 亿元，实际为 5,520 万元）
2. **混淆事实**：将公开信息与招股说明书数据混淆（如技术标准名称、上游企业名单）
3. **无法溯源**：回答无来源依据，无法验证正确性

- RAG 平均 4.38s（含检索 0.3s + Query 理解 1.4s + LLM 生成 1.5s + 模型冷加载）

3. 可考虑增加 BM25 权重或引入更多检索策略提升召回质量
